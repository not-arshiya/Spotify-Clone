# Melodify
A modern music streaming web application inspired by the user experience of popular music streaming platforms. Built as a front-end development project using HTML, CSS, and JavaScript.

# Features
- Responsive music streaming interface
- Play, pause, next, and previous controls
- Dynamic playlist loading
- Album browsing
- Interactive seek bar
- Volume control
- Responsive design for desktop and mobile devices

# Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

# Project Structure
melodify/
├── css/
├── img/
├── js/
├── songs/
├── index.html
└── README.md

# Purpose
This project was developed to strengthen front-end development skills by recreating the user experience of a modern music streaming application. The focus was on responsive design, DOM manipulation, JavaScript functionality, and creating a smooth user interface.
